Ну погоди!
============

![Скриншот](./files/screen-shot.png)

Любимая игрушка в любом компьютере, в котором установлена Java SE версии 1.6 или новее. Игра управляется клавиатурой:

- Пауза - клавиша пробела.
- **F** - волк ловит слева вверху
- **V** - снизу слева
- **J** - правый верхний
- **N** - снизу справа
